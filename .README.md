## Features - 產品功能

- 使用者可以輸入網址
- 使用者點擊 Shorten 後可以產生短網址
- 使用者可以透過點擊 Copy 複製短網址，並跳出複製成功訊息
- 使用者輸入短網址可跳轉回原網址
- 使用者若輸入同樣的網址，將會產生同樣的短網址

## Installation - 專案安裝流程

1. 請先確認有安裝 node.js, npm, nodemon
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，安裝：

```
npm install
npm i nodemon
```

4. Start the server

```
npm run dev
```

5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

```
App is running on http://localhost:30003
```

6. 若欲暫停使用

```
ctrl + c
```
